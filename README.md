## ðŸ”„ Core Loop

**Night Session (18:00):**
- Buy stores
- Upgrade stores
- Protect from thugs
- Restock supplies
- Earn achievements & rewards
- Collect coins & items

**Morning Session (06:00):**
- Market closed
- Buy materials for restocking
- Explore the map to collect upgrade materials
